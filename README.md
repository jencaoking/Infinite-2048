# Infinite-2048 小米手环游戏

基于 Xiaomi Vela JS 开发的2048游戏，专为小米手环10（1.72英寸，212×520分辨率）优化。

## 游戏特点

- **无上限分数**: 游戏分数累加无上限，挑战你的极限
- **胜利条件**: 当任意方块达到20480时触发胜利提示
- **流畅操作**: 支持上下左右滑动操作，响应灵敏
- **视觉效果**: 方块合并动画，颜色渐变系统
- **性能优化**: 针对小米手环硬件进行优化，电池友好
- **完全适配**: 1.72英寸屏幕，212×520分辨率完美适配

## 技术栈

- **框架**: Xiaomi Vela JS (声明式UI + MVVM范式)
- **语言**: JavaScript
- **设备**: 小米手环10及以上
- **最小平台版本**: 1000

## 游戏玩法

1. **滑动操作**: 在游戏面板上滑动手指，控制方块向四个方向移动
2. **合并规则**: 相同数值的方块会合并成它们的和，合并时会获得相应分数
3. **胜利条件**: 当任意方块达到20480时，游戏会显示胜利提示
4. **重新开始**: 点击底部"重新开始"按钮可以重置游戏
